# Развертывание на PythonAnywhere

## Шаги для развертывания:

### 1. Создайте аккаунт на PythonAnywhere
- Перейдите на [pythonanywhere.com](https://www.pythonanywhere.com/)
- Зарегистрируйтесь с помощью email (например, glebtokar10b@gmail.com)
- Подтвердите аккаунт через ссылку в письме

### 2. Загрузите файлы бота
- В личном кабинете PythonAnywhere перейдите в раздел "Files"
- Создайте новую папку `uni_schedule_bot` в вашем домашнем каталоге
- Загрузите все файлы из папки проекта (bot.py, requirements.txt, schedule.json, .env) в созданную папку

### 3. Установите зависимости
- Перейдите в раздел "Consoles"
- Создайте новую консоль (Bash)
- Запустите команду:
  ```bash
  pip install -r /home/your_username/uni_schedule_bot/requirements.txt --user
  ```

### 4. Настройте переменные окружения
- В разделе "Files" откройте файл `.env`
- Убедитесь, что он содержит правильный токен:
  ```
  BOT_TOKEN=8056980600:AAFEzMofmYqpOPVCfX_lWMUIbvDxauN3lRY
  ```

### 5. Создайте задачу для запуска бота
- Перейдите в раздел "Tasks"
- Нажмите "Create a new task"
- Укажите команду для запуска:
  ```
  python3 /home/your_username/uni_schedule_bot/bot.py
  ```
- Установите интервал запуска (например, раз в 24 часа)
- Нажмите "Create"

### 6. Проверка работы бота
- Перейдите в раздел "Consoles"
- Создайте новую консоль (Bash)
- Запустите бота вручную для проверки:
  ```bash
  cd /home/your_username/uni_schedule_bot && python3 bot.py
  ```
- Откройте Telegram и отправьте боту команду `/start`

### 7. Настройте логирование (необязательно)
- Для просмотра логов бота используйте раздел "Files" и проверяйте содержимое файла `bot.log`
- Или используйте команду в консоли:
  ```bash
  tail -f /home/your_username/uni_schedule_bot/bot.log
  ```

## Важные заметки:
- На бесплатном тарифе PythonAnywhere ограничивает ресурсы (CPU, память), но для простого бота этого достаточно
- Если бот перестает работать, проверьте логи и обновите задачу в разделе "Tasks"
- Убедитесь, что файл `schedule.json` содержит актуальные данные о расписании
